What you're doing

Hand-labeling the truth value for each (beach, field) cell. The LLM extracted ~3 variants per cell from one or more source URLs; your job is to pick the right answer (or write one). The result becomes the gold set the calibrator scores variants against.

Workflow — click the variant whose source you trust most. That copies its value into the truth field and stamps the source URL. For enums, the value snaps to the enum if valid; for prose, edit freely. You can also Exclude a cell entirely (button on the card) when no source is trustworthy or the field doesn't apply.

Confidence rubric

highOperator's official site says it directly. No ambiguity. mediumInferred from operator content, or sources mostly agree. Some judgment applied. lowSource thin or only third-party (CVB, blog). Best guess.

Format conventions — for hours_text use 5am-10pm / 5:30am-10pm / 24h / unclear (lowercase, no spaces, hyphen). Free-text fields should be 1–2 specific sentences, no marketing fluff.

Trust hierarchy when sources disagree: operator's official site (parks.ca.gov, city.gov, nps.gov) → curated tourism (visit___.com) → aggregator/blog. Pick the most authoritative; mark confidence accordingly.
